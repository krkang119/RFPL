## 사이트 주소
배포 후 자동 생성: `https://[github계정명].github.io/rfpl`

## 수정 방법
`index.html` 파일을 열어 아래 항목을 업데이트하세요:
- 연구원 이름 및 연구 주제 (`#members` 섹션)
- 논문 및 프로젝트 목록 (`#publications` 섹션)
- 교수님 사진 (`prof-photo` 부분)
- 연구실 호수 및 위치 (`#contact` 섹션)
